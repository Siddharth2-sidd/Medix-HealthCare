# 🏥 Medix-HealthCare

Medix-HealthCare is a **full-stack healthcare claim processing system** designed to manage insurance policies, claims, hospitals, and pharmacies efficiently.

The project is built using **ASP.NET Core Web API for the backend** and **React (Vite) for the frontend**, following a clean and scalable architecture.

---

## 🚀 Tech Stack

### Backend

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* JWT Authentication
* AutoMapper

### Frontend

* React (Vite)
* Axios
* React Router
* CSS / Bootstrap

### Tools

* Visual Studio
* VS Code
* Git & GitHub

---


## ✨ Features

### 🔐 Authentication

* User Signup
* User Login
* JWT Authentication
* Role-based authorization

### 👨‍⚕️ Admin Features

* Manage policies
* Manage hospitals
* Manage pharmacies
* Manage claims

### 🏥 Hospital

* Submit insurance claims
* Track claim status

### 💊 Pharmacy

* Manage medicines
* Process insurance claims

### 📄 Claims Management

* Submit claim
* Approve / Reject claim
* Claim history tracking
